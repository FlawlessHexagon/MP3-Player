Audio File Naming

The player reads track title and artist directly from the filename. No metadata tags are used. Files must follow this naming format:

Artist Name - Track Title.mp3

Examples:

Daft Punk - Get Lucky.mp3
Aphex Twin - Avril 14th.flac
The Prodigy - Firestarter.wav

The part before the first " - " (space-hyphen-space) becomes the artist. Everything after becomes the title. If no " - " is found, the entire filename (without extension) is used as the title and the artist is set to "Unknown Artist".

Supported formats: MP3, FLAC, OGG, WAV, AAC, M4A.

Playlist File Format

Each playlist is a single .json file. The filename (without .json) becomes the playlist name inside the app.

A playlist file contains a flat JSON array of track IDs. Track IDs are generated when you load your music folder — they are not filenames. To build playlists that persist across sessions, use the in-app workflow: load your music, add tracks to a playlist via right-click → "Add to Playlist", then export each playlist individually using the export icon next to its name in the sidebar.

Example file — "Chill Evening.json":

[
  "a3f9bc12",
  "d84e1f77",
  "cc029a31"
]

Alternatively, a playlist file can contain an object where each key is a playlist name and each value is an array of IDs. This is the format produced by "Export All as JSON":

{
  "Chill Evening": ["a3f9bc12", "d84e1f77"],
  "Workout": ["bb3312ef", "90ac44de"]
}

To load playlists:

  • Desktop: Settings → Playlists → Open Playlist Folder (selects a folder of .json files, one per playlist)
  • All browsers: Settings → Playlists → Select Playlist Files (pick one or more .json files manually)

Using the Player

Loading music:

  • Desktop (Chrome/Edge): Settings → Music Source → Open Folder. The app scans the folder recursively including subfolders.
  • All browsers: Settings → Music Source → Select Files. Pick individual audio files.
  • Recently opened folders are remembered and can be reloaded in one click from the Recent Folders list.

Playing tracks:

  • Double-click a track to play it immediately.
  • Single-click to select. Right-click (or long-press on mobile) for options.
  • Use Cmd/Ctrl+Click to select multiple tracks. Shift+Click to select a range.

Playlists:

  • Click the + button next to PLAYLISTS in the sidebar to create a new playlist.
  • Select tracks, right-click → Add to Playlist to add them.
  • Click the export icon next to a playlist name to download it as a .json file.
  • To remove a track from a playlist, navigate to that playlist, right-click the track → Remove from Playlist.

Controls:

  • Shuffle: randomises the play queue. Click again to restore order.
  • Repeat: cycles between Off, Repeat One, and Repeat All.
  • Seek bar: click or drag to jump to any point in the track.
  • Volume: drag the slider on the right of the player bar.

Mobile:

  • Tap the player bar to open the full Now Playing sheet.
  • Swipe down on the sheet to dismiss it.
  • Swipe left or right on the sheet to skip tracks.
  • Long-press a track to enter select mode with checkboxes.
